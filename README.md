# PWM-shift-register
This project demonstrates PWM signal generation using a shift register by cycling binary patterns to control duty cycle. It highlights sequential digital logic concepts and is suitable for applications such as LED brightness control and basic motor speed regulation.
 PWM Generator using Verilog HDL

This project implements a basic Pulse Width Modulation (PWM) generator using Verilog HDL. The PWM output is generated by varying the duty cycle, which controls how long the signal stays HIGH in each cycle.

## Files
-PWM_Gen.v – Verilog code for PWM generation  
- PWM_Test.v – Testbench for simulating the PWM output

## Tools Used
- Verilog HDL  
- Xilinx ISE

## Applications
- LED brightness control  
- Motor speed control
